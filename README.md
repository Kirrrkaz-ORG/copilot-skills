# 🧠 Copilot Skills Library

Персональная библиотека навыков для **GitHub Copilot** — набор готовых `SKILL.md` файлов, которые автоматически подхватываются агентом и расширяют его возможности.

> Навыки работают **во всех рабочих пространствах** — установил один раз и забыл.

## 🚀 Установка навыка — пошагово

> Делается один раз. Потом навык работает сам.

### Способ 1 — Вручную через Проводник (проще всего)

**Шаг 1.** Скачай этот репозиторий как ZIP:
- Нажми зелёную кнопку **Code** → **Download ZIP**
- Распакуй архив в любую удобную папку

**Шаг 2.** Открой папку, куда нужно скопировать навык.

На **Windows** — вставь этот путь в адресную строку Проводника:
```
%USERPROFILE%\.copilot\skills
```
> Если папка `.copilot\skills` не существует — создай её вручную.
> В Проводнике: ПКМ → Создать папку → `.copilot`, внутри ещё одну — `skills`.

На **macOS/Linux** — путь:
```
~/.copilot/skills/
```

**Шаг 3.** Скопируй папку навыка целиком:
- Возьми нужную папку из распакованного архива (они лежат в `skills/`)
- Вставь её в папку `skills` из шага 2

Должно получиться вот так:
```
C:\Users\ИМЯ_ПОЛЬЗОВАТЕЛЯ\.copilot\skills\
    ├── pdf-table-digitize\
    │   ├── SKILL.md
    │   ├── README.md
    │   ├── scripts\
    │   └── references\
    ├── journal-to-vedomost\
    │   ├── SKILL.md
    │   └── README.md
    └── exam-tickets-generator\
        ├── SKILL.md
        └── README.md
```

**Шаг 4.** Перезапусти VS Code — навыки появятся автоматически.

---

### Способ 2 — Через терминал (быстрее)

```powershell
# Windows (PowerShell) — запусти из папки с распакованным архивом
Copy-Item -Recurse ".\skills\*" "$env:USERPROFILE\.copilot\skills\"
```

```bash
# macOS / Linux — запусти из папки с распакованным архивом
cp -r ./skills/* ~/.copilot/skills/
```

### Как использовать после установки

Открой чат Copilot в VS Code и просто напиши запрос — навык загрузится сам:

**Оцифровка таблиц из PDF:**
> `оцифруй таблицу из этого pdf`  
> `перевести pdf в excel`

**Генерация зачётных ведомостей:**
> `создай зачётную ведомость из журнала`  
> `перенести оценки в ведомость`

**Генерация экзаменационных билетов:**
> `создай билеты по шаблону`  
> `сгенерируй эталонные ответы`

## 🛠️ Добавить новый навык

1. Создай папку `skills/<имя-навыка>/`
2. Добавь `SKILL.md` с frontmatter:
   ```yaml
   ---
   name: имя-навыка
   description: 'Что делает и когда использовать. Триггеры: ...'
   ---
   ```
3. **⚠️ ОБЯЗАТЕЛЬНО:** Обнови `README.md`:
   - Добавь строку в таблицу навыков
   - Добавь описание в секцию "Особенности навыков"
   - Обнови структуру репозитория
4. Скопируй в `~/.copilot/skills/` и открой PR

> 🔴 **Правило:** Без обновления README.md — PR не принимается!

## 📝 Особенности навыков

### pdf-table-digitize
- Распознавание таблиц из отсканированных PDF
- Сохранение структуры с объединёнными ячейками
- Экспорт в Markdown и Excel (.xlsx)
- Работает с документами без текстового слоя

### journal-to-vedomost
- Распознавание оценок из скриншотов журналов
- Автоматическое формирование ведомости по шаблону
- Поддержка различных систем журналов
- Генерация Word документов с правильным форматированием

### exam-tickets-generator
- Парсинг вопросов из Markdown или Word
- Генерация билетов с таблицей-шапкой по образцу
- Создание документов с эталонными ответами
- Автоматическое устранение дубликатов
- Поддержка теоретических вопросов и практических заданий

### loom-tz-analyzer
- Извлечение технических заданий из Loom видео (через API или вручную)
- Структурирование требований: цель, фичи, стек, сроки, MVP scope
- Автоматическое выделение open questions
- Поддержка транскриптов на любом языке
- Создание документации в формате Markdown с чеклистами
