# 🧠 Copilot Skills Library

ерсональная библиотека навыков для **GitHub Copilot** — набор готовых `SKILL.md` файлов, которые автоматически подхватываются агентом и расширяют его возможности.

> авыки работают **во всех рабочих пространствах** — установил один раз и забыл.

## 🚀 становка навыка — пошагово

> елается один раз. отом навык работает сам.

### Способ 1 — ручную через роводник (проще всего)

**Шаг 1.** Скачай этот репозиторий как ZIP:
- ажми зелёную кнопку **Code** → **Download ZIP**
- аспакуй архив в любую удобную папку

**Шаг 2.** ткрой папку, куда нужно скопировать навык.

а **Windows** — вставь этот путь в адресную строку роводника:
```
%USERPROFILE%\.copilot\skills
```
> сли папка `.copilot\skills` не существует — создай её вручную.
>  роводнике:  → Создать папку → `.copilot`, внутри ещё одну — `skills`.

а **macOS/Linux** — путь:
```
~/.copilot/skills/
```

**Шаг 3.** Скопируй папку навыка целиком:
- озьми нужную папку из распакованного архива (они лежат в `skills/`)
- ставь её в папку `skills` из шага 2

олжно получиться вот так:
```
C:\Users\Я_ЬТЯ\.copilot\skills\
    ├── pdf-table-digitize\
    │   ├── SKILL.md
    │   ├── README.md
    │   ├── scripts\
    │   └── references\
    ├── journal-to-vedomost\
    │   ├── SKILL.md
    │   └── README.md
    └── exam-tickets-generator\
        ├── SKILL.md
        └── README.md
```

**Шаг 4.** ерезапусти VS Code — навыки появятся автоматически.

### Способ 2 — ерез терминал (быстрее)

`powershell
# Windows (PowerShell) — запусти из папки с распакованным архивом
Copy-Item -Recurse ".\skills\*" "$env:USERPROFILE\.copilot\skills\"
`

`ash
# macOS / Linux — запусти из папки с распакованным архивом
cp -r ./skills/* ~/.copilot/skills/
`

### ак использовать после установки

ткрой чат Copilot в VS Code и просто напиши запрос — навык загрузится сам:

**цифровка таблиц из PDF:**
> `оцифруй таблицу из этого pdf`  
> `перевести pdf в excel`

**енерация зачётных ведомостей:**
> `создай зачётную ведомость из журнала`  
> `перенести оценки в ведомость`

**енерация экзаменационных билетов:**
> `создай билеты по шаблону`  
> `сгенерируй эталонные ответы`

```

    ├── pdf-table-digitize/             # авык 1: цифровка таблиц
    │   ├── SKILL.md                    # сновные инструкции

    ├── journal-to-vedomost/            # авык 2: ачётные ведомости
    │   ├── SKILL.md                    # роцедура генерации
    │   └── README.md
    └── exam-tickets-generator/         # авык 3: кзаменационные билеты
        ├── SKILL.md                    # олная документация
        └── README.md                   # раткое описание
```

## 🛠️ обавить новый навык

1. Создай папку `skills/<имя-навыка>/`
2. обавь `SKILL.md` с frontmatter:
   `yaml
   ---
   name: имя-навыка
   description: 'то делает и когда использовать. Триггеры: ...'
   ---
   `
3. Скопируй в `~/.copilot/skills/` и открой PR

## 📝 собенности навыков

### pdf-table-digitize
- аспознавание таблиц из отсканированных PDF
- Сохранение структуры с объединёнными ячейками
- кспорт в Markdown и Excel (.xlsx)
- аботает с документами без текстового слоя

### journal-to-vedomost
- аспознавание оценок из скриншотов журналов
- втоматическое формирование ведомости по шаблону
- оддержка различных систем журналов
- енерация Word документов с правильным форматированием

### exam-tickets-generator
- арсинг вопросов из Markdown или Word
- енерация билетов с таблицей-шапкой по образцу
- Создание документов с эталонными ответами
- втоматическое устранение дубликатов
- оддержка теоретических вопросов и практических заданий
