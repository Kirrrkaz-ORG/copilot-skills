# Справочник: Паттерны оцифровки таблиц из PDF

## Типичные структуры таблиц нагрузки

### Двухуровневая шапка (merge + подзаголовки)

```
┌─────────────┬──────────────────┬─────────────┐
│ ФИО         │  Кол-во часов    │   Всего     │
│             ├────────┬─────────┤             │
│             │ 1 п/г  │ 2 п/г  │             │
├─────────────┼────────┼─────────┼─────────────┤
│ Иванов И.И. │   34   │   36   │     70      │
└─────────────┴────────┴─────────┴─────────────┘
```

openpyxl:
```python
ws.merge_cells("C2:D2")   # "Кол-во часов" — объединить строку
ws.merge_cells("A2:A3")   # "ФИО" — объединить строки
ws.merge_cells("E2:E3")   # "Всего" — объединить строки
# Подзаголовки в строке 3:
ws.cell(3, 3).value = "1 п/г"
ws.cell(3, 4).value = "2 п/г"
```

## Объединённые строки (вертикальный merge)

Когда несколько строк относятся к одному сотруднику/разделу:

```python
# Данные идут с 4-й строки, 12 строк всего
ws.merge_cells("A4:A15")   # ФИО
ws.cell(4, 1).value = "Иванов И.И."
ws.cell(4, 1).alignment = Alignment(horizontal="center", vertical="center", wrap_text=True)
```

## Ориентация страниц в PDF

| Хранение в PDF | Нужный поворот Pillow | Параметр rotate |
|---|---|---|
| Портрет (данные читаются нормально) | Не нужен | — |
| Портрет, но данные ландшафт (CW) | Повернуть CCW | `rotate(90, expand=True)` |
| Портрет, но данные ландшафт (CCW) | Повернуть CW | `rotate(-90, expand=True)` |

Определить поворот можно визуально: открыть PNG в VS Code и проверить, в какую сторону повёрнут текст.

## Нарезка PNG для анализа

```python
from PIL import Image

img = Image.open("page_1.png")
sections = {
    "header": (0, 0, img.width, 500),        # Шапка
    "rows":   (0, 500, img.width, 1400),      # Данные
    "itogo":  (0, 1400, img.width, img.height), # ИТОГО
}
for name, box in sections.items():
    img.crop(box).save(f"sec_{name}.png")
```

## Проверочный список перед созданием .xlsx

- [ ] Все столбцы правильно подсчитаны (не перепутаны 1п/г и 2п/г)
- [ ] Объединения ФИО/дисциплины корректны (merge не перекрывает числовые ячейки)
- [ ] Строка ИТОГО совпадает с суммой столбцов
- [ ] Ставка 0,5 хранится как число (не строка "0,5")
- [ ] Ширины столбцов установлены (длинные тексты — wrap_text=True)
- [ ] Заморозка заголовков: `ws.freeze_panes = "A4"`
- [ ] Временные PNG удалены после завершения
