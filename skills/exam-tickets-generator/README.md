# Генератор экзаменационных билетов

Автоматическая генерация экзаменационных билетов и эталонных ответов в формате Word (.docx) на основе шаблонов.

## Описание

Навык позволяет быстро создавать профессионально оформленные экзаменационные билеты для образовательных учреждений. Поддерживает парсинг вопросов из различных форматов и генерацию документов по утверждённым стандартам.

## Возможности

- 📝 Парсинг вопросов из Markdown и Word документов
- 📄 Генерация билетов с таблицей-шапкой по шаблону
- ✅ Создание документов с эталонными ответами
- 🔄 Автоматическое устранение дубликатов вопросов
- 📊 Поддержка теоретических вопросов и практических заданий
- 🎯 Гибкая адаптация под требования учреждения

## Когда использовать

- Подготовка КИМ (контрольно-измерительных материалов)
- Создание экзаменационных билетов по образцу
- Генерация приложений с эталонными ответами
- Автоматизация документооборота в образовании

## Требования

- Python 3.8+
- Библиотека `python-docx`

```bash
pip install python-docx
```

## Быстрый старт

1. Подготовьте файл с вопросами в формате Markdown
2. Предоставьте образец существующего билета для референса
3. Запросите генерацию билетов и эталонных ответов

Подробная документация находится в [SKILL.md](SKILL.md).

## Пример использования

**Входные данные (Markdown):**
```markdown
**Билет №1**

**Теоретические вопросы:**
1. Первый вопрос?
2. Второй вопрос?

**Практические задания:**
1. Задание 1...
2. Задание 2...
```

**Результат:** Word документы с билетами и эталонными ответами в утверждённом формате.
