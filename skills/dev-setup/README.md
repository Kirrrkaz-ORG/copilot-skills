# 🚀 Dev Environment Setup Skill

Автоматизация полного setup локального dev окружения для **Next.js + Supabase** проектов.

## 📋 Что делает

Этот навык проведёт через все шаги настройки:

1. ✅ **Создание Supabase проекта** для локальной разработки
2. ✅ **Настройка `.env.local`** с API ключами
3. ✅ **Запуск миграций** базы данных
4. ✅ **Загрузка seed данных** (тестовый контент)
5. ✅ **Создание тестового пользователя** для логина
6. ✅ **Проверка** что всё работает

## 🎯 Когда использовать

- 🆕 Настройка проекта на новой машине
- 🔄 Сброс локального окружения до чистого состояния
- 👥 Onboarding нового разработчика
- 🛡️ Создание изолированной dev базы (отдельно от production)

## 🚫 Когда НЕ использовать

- Production deployment → используй Vercel/деплой инструменты
- Просто установка зависимостей → запусти `npm install`
- Debugging → используй логи/errors

## 🆘 Troubleshooting

Все типичные проблемы и решения описаны в `SKILL.md` → раздел **Troubleshooting**.

Частые вопросы:

**Q: Почему данные не появляются на localhost?**  
A: Localhost использует НОВУЮ пустую dev базу. Нужно загрузить seed данные (шаг 6).

**Q: "Invalid API key" при запуске**  
A: Перезапусти dev сервер (`Ctrl+C` → `npm run dev`) чтобы перезагрузить `.env.local`.

**Q: Email rate limit при создании пользователя**  
A: Используй Admin API (шаг 7.1) вместо обычного signup.
